# Online Retail Application Database

## Project Description
This project is a database management system for an online retail store using MySQL.

The system manages:
- Customers
- Products
- Orders
- Payments
- Categories

---

## Technologies Used
- MySQL
- MySQL Workbench

---

## Features
- Customer Management
- Product Management
- Order Tracking
- Payment Records
- Sales Reports
- Stored Procedures

---

## Database Tables
1. Customers
2. Products
3. Orders
4. Payments
5. Categories
6. Order_Items

---

## SQL Concepts Used
- CREATE TABLE
- PRIMARY KEY
- FOREIGN KEY
- JOINS
- STORED PROCEDURES
- AGGREGATE FUNCTIONS
